# CASO-PRACTICO-EXCEL-
Este repositorio contiene un analisis de datos realizados en excel a partir de un conjunto de transacciones, el objetivo principal fue identificar patrones de venta, productos mas vendidos y comportamiento por sucursal
ANALISIS REALIZADO
Conteo de transacciones por producto, suma total de ventas por producto, ranking de productos mas vendidos 
algunos de los resultados obtenidos fue que JUNIO fue el mes con mayor ingresos 
COFFE fue la categoria con mas numero de transacciones 
El producto sustainably grown organic fue el que obtuvo mayor suma de ingresos
